A lightweight, static landing page clone of the Amazon Prime Video interface. This project was built to practice modern CSS layouts, focusing on Flexbox and UI transitions.

## ✨ Key Features
* **Full Flexbox Layout:** Used Flexbox for the navigation bar, feature rows, FAQ section, and footer.
* **Welcome Section:** Implementation of a layered background with a dark overlay and centered typography.
* **Smooth Transitions:** Added `ease-out` hover effects and button animations for a premium feel.
* **Responsive Design:** Utilized `object-fit: cover` and `object-position` for image handling, with initial Media Queries for cross-device support.

## 🛠️ Technical Implementation
* **HTML5:** Semantic structure for better accessibility.
* **CSS3:** Custom styling without external libraries or frameworks.
* **Typography:** Integrated 'Inter' and sans-serif fallbacks to mimic the 'Amazon Ember' look.

## 🧠 What I Learned
Building this was a "hell of a ride" in terms of debugging! 😅 I gained deep insights into:
* How to prevent Flexbox from squishing images in narrow containers.
* The importance of the **LVHA** (Link, Visited, Hover, Active) order in CSS.
* Using `flex-direction: column-reverse` to flip layout order on mobile devices.
